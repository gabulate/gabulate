Jake Doe's Music & Merch

Co-Authored by Melanny Vargas


This application is an ECommerce / MarketPlace type of website, themed for music products.
It was made for a programming course where we had to create a website with open source tools using Angular for the front-end, a Node.js back-end server and MySQL with Prisma ORM as database.

The users are shown a grid of available products to purchase. These can be filtered by category and sorted by price, or the user can type in the search bar to look up a specific product.


The product view shows the selected product with more detail, displaying multiple images and information such as the item's description, category, seller, available quantity and condition. On the side is a list of similar products the user might be interested in.
The user has the option to add the product to their cart and procced with the purchase.


Further into the product's page, the user can see comments and questions made by other users and is given the option to make one as well. The seller can respond to these comments and they will be shown to everyone.


Sellers and administrators can look at a dashboard that shows a report of their best selling products and a pie chart of user ratings.
Administrators can also see a table that displays the best and worst rated sellers.


Features for sellers:

  • Option to add / edit / remove products.
  • List of all placed orders.
  • Detailed view of placed orders.
  • Option to update orders' state.
  • Option to respond to comments.
  • Dashboard analytics view of products.

Features for buyers are:

  • Product cart.
  • Functionality to save multiple delivery directions.
  • Functionality to save multiple payment methods.
  • Purchase history.
  • Order state information.
  • Product rating.

Security features include:

  • Functionality to register and login users.
  • Encripted passwords.
  • User roles: Some pages can only be seen by users with an specific role.
  • Functionality to de-activate users by administrators.
